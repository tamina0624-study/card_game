# docs/ 索引

このゲームの仕様・設計・進捗・画面リファレンスをまとめたフォルダ。ルートの[README.md](../README.md)からは主要ドキュメントのみリンクしているため、詳細はここから辿る。

## 仕様・設計

| ファイル | 内容 |
|---|---|
| [開発指示書.md](./開発指示書.md) | ゲーム仕様全体・確定事項(カード作成・デッキ・必殺技・AI戦闘システム・JSON出力形式) |
| [システムプロンプト.md](./システムプロンプト.md) | AI審判(アルゼリオン)用システムプロンプト。`src/lib/claude/client.ts`が実行時に読み込む |
| [設計.md](./設計.md) | アーキテクチャ・DBスキーマ・APIルート設計 |
| [画面構成・UI部品名.md](./画面構成・UI部品名.md) | 画面・画面内領域・UIコンポーネントの命名リファレンス(画面キャプチャ付き) |
| [進捗.md](./進捗.md) | 実装タスクの進捗・検証記録 |

## サブフォルダ

## 更新の目安

- ゲーム仕様そのものを変えたら → 開発指示書.md
- AIの審判挙動を変えたら → システムプロンプト.md
- DBスキーマ・APIルートを変えたら → 設計.md
- 画面のレイアウト・命名を変えたら → 画面構成・UI部品名.md(必要ならキャプチャも撮り直す)
- タスクを進めたら → 進捗.md
